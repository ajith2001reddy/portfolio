# Ajith Pavan Reddy | UX Writing Portfolio

This is the source code for my professional UX Writing portfolio.

## 🚀 Technical Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Vanilla)
- **Interactions:** JavaScript (ES6+)
- **Icons:** Lucide Icons

## 📂 Project Structure
- `index.html`: Main landing page
- `case-study-tenant.html`: Property Management case study
- `case-study-chat.html`: Real-time Chat case study
- `case-study-tcs.html`: TCS Infrastructure case study
- `style.css`: Global design system and animations
- `script.js`: Core logic for reveal animations and micro-interactions

## 🛠️ Local Development
Simply open `index.html` in any modern web browser.
